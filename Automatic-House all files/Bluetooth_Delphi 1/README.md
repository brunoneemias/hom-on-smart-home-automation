# 📡 Bluetooth Delphi — Controle via HC-05

> Projeto Delphi de controle Bluetooth com Arduino usando o módulo HC-05, utilizado como base de estudos para o desenvolvimento do app **hom\<on>**.

---

## 📖 Sobre

Esta pasta contém um projeto de referência em **Delphi XE10** para comunicação Bluetooth com o módulo **HC-05**, incluindo esquemas de circuito e documentação de apoio usada durante o desenvolvimento do sistema de automação.

---

## 📁 Arquivos

| Arquivo | Descrição |
|---|---|
| `BluetoothRControl.dproj` | Projeto principal Delphi |
| `Display.pas` / `Display.fmx` | Tela e lógica de controle |
| `Bluetooth.fzz` | Esquema de circuito (Fritzing) |
| `Fritz.JPG` | Imagem do esquema elétrico |
| `Bluetooth_Arduino/` | Código Arduino para comunicação |
| `AndroidManifest.template.xml` | Configurações Android |
| `DELPHİ XE10-HC05_ENGLISH.docx` | Documentação em inglês |
| `DELPHİ XE10-HC05_TURKCE.docx` | Documentação em turco |

---

## 🛠️ Tecnologias

- **Delphi XE10** + FireMonkey
- **Módulo Bluetooth HC-05**
- **Arduino**
- **Fritzing** (diagrama do circuito)

---

## 📌 Observação

Este módulo foi utilizado como **referência técnica** durante o desenvolvimento do app principal **hom\<on>**, auxiliando na implementação da comunicação Bluetooth entre o aplicativo Android e o Arduino.
